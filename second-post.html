Second Post

Date Sun 18 November 2018

This is the second post of a series of posts. It shows an embedded youtube video.

To do so I followed the tutorial. Since some of the materials there are out of date, so I explain a little bit in the following. You could use the plugin liquid_tags.youtube in pelicanconf.py file.

#pelicanconf.py

PLUGINS = [
    'liquid_tags.youtube']

And then include the following line in your post second_post.md.

{% youtube Qq-5frjUfK0 [640] [390] %}

Notice here the difference between the one in the tutorial and the above is that I only write the video id for the youtube video instead of the full https address.
